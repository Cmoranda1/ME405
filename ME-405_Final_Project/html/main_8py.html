def main.start_up_calibration ()
 This function is not for a task it must be done before the tasks begin running, to ensure accurate values from the sensors Using the serial terminal, the function will request that the table is level before beginning The pin PA6 will then be turned on to power up the sensor, and therefore reset the Euler Angles to 0 before beginning.
